SCA-IT: Industrial Infrastructure Platform
SCA-IT is a professional full-stack web application tailored for IT infrastructure and industrial solutions providers. It showcases high-end services like active networking, data center solutions, and surveillance systems through a modern, cyber-industrial interface.

🚀 Features
Premium Design: Implements a "Midnight Stellar" theme with glassmorphic UI elements for a high-tech aesthetic.

Service Portfolio: Dedicated sections for Networking, Data Centers, CCTV/Surveillance, and Access Control.

Interactive UX: Smooth transitions, hover effects, and custom scanline animations.

Full-Stack Architecture: Built using the MERN stack for scalable data management.

Mobile Responsive: Fully optimized for seamless browsing across all device types.

🛠️ Tech Stack
Frontend: React.js, Vite, tailwindCss

Backend: Java-script, Node.js, Express.js

Database: MongoDB Atlas

Deployment: Vercel (Frontend), Render (Backend)

📦 Installation
Clone the repo: git clone https://github.com/shubhamsoni03s/SCA-IT.git

Setup Backend: Run npm install in the /server folder and add your .env variables.

Setup Frontend: Run npm install and npm run dev in the /client folder.
